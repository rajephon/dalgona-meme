# Dalgona meme generator

[dalgona.games](https://dalgona.games)

# Usage

## Install

```shell
git clone git@github.com:rajephon/dalgona-meme.git
cd dalgona-meme
yarn install # or npm instlal
```

## Run

just start.

```shell
yarn run start # or npm run start
```
